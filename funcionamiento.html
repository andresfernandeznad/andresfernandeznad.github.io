FUNCIONAMIENTO
Al pulsar una tecla esta genera un pulso eléctrico y que se manda a un microcontrolador, el que se encarga a determinar que tecla se ha pulsado.
Una vez determinada la tecla, el microcontrolador genera un código que se llama Scan Code y si son varias teclas genera un Break Code.
Estos códigos son manejados por una aplicación BIOS, administrador de teclado, la que compara los códigos con una tabla y los manda al microprocesador.
